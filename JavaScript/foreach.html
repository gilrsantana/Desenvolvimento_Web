Estrutura de repetição 'forEach'

Essa estrutura de repetição também percorre um array semelhante as outra estruturas for e for in, mas com algumas peculiaridades.

  1. Ela só considera os índices do array que sejam maior ou igua a zero
  2. Precisa de uma função de calbak que tenha, nesta ordem, as referências ao: valor, indice, array

Implantação da estrutura forEach():
var funcionario = Array()

  • funcionario[0] = 'Miguel'
  • funcionario[1] = 'Antonio'
  • funcionario[2] = 'Valber'
  • funcionario[3] = 'Carlos'
  • funcionario[4] = 'Kely'

atribuição da função auxiliar:
var aux = function(valor, indice, array) {
document.write('indice: ' + indice + ' | valor: ' + valor)
if(valor == 'Miguel'){
array[indice] = 'Sandra'
}
}

utilizando forEach():
funcionario.forEach(aux)

Ou seja, o 'forEach()' percorre um array e faz com que para cada elemento deste array seja aplicada uma função ou comportamento
