Estrutura de repetição 'for in'

Essa estrutura de repetição tem o mesmo comportamento que o 'for' só que com uma inteligência a mais.

Como um array pode assumir índices dos mais variados tipos, exemplo:

  • array[2] - índice inteiro positivo
  • array[-4] - índice inteiro negativo
  • array['5'] - índice string numérica
  • array['m'] - índice char
  • array['mosquito'] - indice string
  • array[true] - índice booleano
um laço de repetição comum 'for' não tem inteligência para tratar esses valores ele não vai operar da forma correta.

Já o for in tem a inteligência para percorrer o array enquanto ele não obter como resposta 'undefined'. Para isso é muito comum ele estar acompanhado do '.length', pois com esse atributo ele vai percorrer o tamanho do array não importando seus índices.

var array = Array()
array[2] - índice inteiro positivo
array[-4] - índice inteiro negativo
array['5'] - índice string numérica
array['m'] - índice char
array['mosquito'] - indice string
array[true] - índice booleano
